Here are some useful comments for LaTeX. Many of these have been designed and/or implemented by Michael Uttmark.

\stopPoint will automatically create a "stop point" note in the protocol.
\C{} and \uL{} will automatically create Celsius and microliter symbols, respectively.
\cite{author} will automatically create a citation from those listed in main.bib.
\helpfulTip{(tip here)} will create a small 'aside' in which you can write helpful advice for future readers of the protocol.
https://www.tablesgenerator.com/ is an incredibly useful tool for creating tables in LaTeX.
https://www.overleaf.com/learn/latex/Inserting_Images is a great guide for how to insert images in LaTeX. Make sure to put all images in the 'images' folder.
